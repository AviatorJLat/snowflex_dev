    Create priv/python/snowflex_dev_worker.py -- the long-running Python process that communicates with Elixir over stdin/stdout using {:packet, 4} framed JSON.

    The script MUST follow this exact structure and order:

    **1. Stdout redirection (FIRST -- before any imports that might print):**
    ```python
    #!/usr/bin/env python3
    """SnowflexDev worker -- Erlang Port bridge to Snowflake via snowflake-connector-python."""
    import sys
    import os
    import struct
    import json
    import threading
    import logging
    import time

    # CRITICAL: Redirect stdout to stderr BEFORE importing snowflake.connector.
    # Any library print() would corrupt the {:packet, 4} binary protocol.
    sys.stdout = sys.stderr

    # Now safe to import libraries that might print to stdout
    import snowflake.connector

    # Configure snowflake connector logging to stderr (which stdout now points to)
    logging.getLogger("snowflake.connector").setLevel(logging.WARNING)
    ```

    **2. Protocol I/O constants:**
    ```python
    PROTO_OUT = sys.__stdout__.buffer  # Raw binary stdout for protocol
    PROTO_IN = sys.stdin.buffer        # Raw binary stdin for protocol
    CHUNK_SIZE = 1000                  # Rows per chunk for large result sets
    ```

    **3. read_message() function:**
    - Read 4-byte header from PROTO_IN. If < 4 bytes, return None (EOF).
    - Unpack as big-endian unsigned int: `struct.unpack(">I", header)[0]`
    - Read exactly `length` bytes (loop until complete, return None if EOF mid-read).
    - Return `json.loads(data)`.

    **4. write_message(msg) function:**
    - Serialize msg to JSON bytes: `json.dumps(msg, default=str).encode("utf-8")`
    - Create header: `struct.pack(">I", len(data))`
    - Write header + data as single call: `PROTO_OUT.write(header + data)`
    - Flush immediately: `PROTO_OUT.flush()`
    IMPORTANT: Concatenate header + data into a single write() call to avoid interleaving.

    **5. build_metadata(description) helper:**
    - Takes cursor.description (list of 7-tuples per PEP 249).
    - Returns dict mapping column_name -> {"type_code": type_code, "display_size": display_size, "internal_size": internal_size, "precision": precision, "scale": scale, "null_ok": null_ok}.
    - Convert type_code to int if it is not already (Snowflake connector uses custom type objects).

    **6. serialize_rows(rows) helper:**
    - Takes list of tuples from cursor.fetchall().
    - Returns list of lists (tuples are not JSON-serializable).
    - Uses `json.dumps(default=str)` in write_message to handle Decimal, datetime, date, time, bytes types.

    **7. execute_query(conn, request_id, sql, params) function:**
    - Create cursor, execute sql with params.
    - If cursor.description is None (DDL/DML with no result set), send ok response with columns=None, rows=[], num_rows=cursor.rowcount, metadata={}.
    - Otherwise: extract columns from cursor.description[i][0], build metadata, fetchall() rows.
    - If len(rows) <= CHUNK_SIZE: send single-shot response with columns, rows (as lists), num_rows, metadata.
    - If len(rows) > CHUNK_SIZE: send chunked_start header (status "ok", chunked: true, payload with columns, total_rows, metadata), then chunk messages (status "chunk", payload with rows slice and chunk_index), then done message (status "done", payload with chunks_sent count).
    - Close cursor after sending all data.

    **8. stdin_monitor_ppid() function (zombie prevention -- PORT-04):**
    ```python
    def stdin_monitor_ppid():
        """Exit if parent process dies. Prevents zombie Python processes."""
        original_ppid = os.getppid()
        while True:
            time.sleep(1)
            if os.getppid() != original_ppid:
                os._exit(1)
    ```

    **9. main() function -- the command loop:**
    - Initialize conn = None.
    - Loop: read_message(). If None, close conn if open, sys.exit(0).
    - Extract request_id and msg_type from message.
    - Handle "connect": call snowflake.connector.connect() with payload fields (account, user, warehouse, database, schema, role, authenticator defaulting to "externalbrowser", login_timeout defaulting to 300). Set client_session_keep_alive=True, client_store_temporary_credential=True. Send ok response on success.
    - Handle "execute": guard conn is not None (send error with code "SNOWFLEX_DEV_001" if not). Call execute_query.
    - Handle "ping": guard conn. Execute "SELECT 1", close cursor, send ok with message "pong".
    - Handle "disconnect": close conn if open, set conn=None, send ok.
    - Handle unknown type: send error with code "SNOWFLEX_DEV_002".
    - Exception handling:
      - snowflake.connector.errors.ProgrammingError: send error with str(e), str(e.errno), e.sqlstate
      - snowflake.connector.errors.DatabaseError: send error with str(e), str(e.errno), getattr(e, 'sqlstate', None)
      - Exception (catch-all): send error with str(e), code "SNOWFLEX_DEV_999"

    **10. Entry point:**
    ```python
    if __name__ == "__main__":
        monitor = threading.Thread(target=stdin_monitor_ppid, daemon=True)
        monitor.start()
        main()
    ```
