    ```elixir
    defmodule SnowflexDev.Transport do
      @moduledoc "Behaviour for transport implementations that communicate with the Python worker."

      @type opts :: keyword()

      @callback start_link(opts()) :: GenServer.on_start()
      @callback execute(pid(), String.t(), list(), keyword()) :: {:ok, SnowflexDev.Result.t()} | {:error, SnowflexDev.Error.t()}
      @callback ping(pid()) :: :ok | {:error, SnowflexDev.Error.t()}
      @callback disconnect(pid()) :: :ok
    end
    ```

    **2. Create lib/snowflex_dev/transport/port.ex -- GenServer implementation:**

    Module: `SnowflexDev.Transport.Port`
    Behaviours: `use GenServer`, `@behaviour SnowflexDev.Transport`

    **State struct** (defmodule State, defstruct inside the module):
    Fields: `port`, `python_path`, `worker_path`, `opts`, `connected` (boolean), `pending_request` (nil | {id, from} | {id, from, :chunking, acc_map})

    **Public API:**
    - `start_link(opts)` -- calls `GenServer.start_link(__MODULE__, opts)`
    - `execute(pid, sql, params \\ [], opts \\ [])` -- calls `GenServer.call(pid, {:execute, sql, params, opts}, query_timeout(opts))`. Default query timeout: 60_000ms.
    - `ping(pid)` -- calls `GenServer.call(pid, :ping, 10_000)`
    - `disconnect(pid)` -- calls `GenServer.call(pid, :disconnect, 5_000)`

    **init/1:**
    - Extract `python_path` from opts, default to `Path.join([File.cwd!(), "_snowflex_dev", "venv", "bin", "python3"])`
    - Set `worker_path` to `Application.app_dir(:snowflex_dev, "priv/python/snowflex_dev_worker.py")`
    - Open Port: `Port.open({:spawn_executable, String.to_charlist(python_path)}, [:binary, {:packet, 4}, :exit_status, :use_stdio, args: ["-u", worker_path]])`
    - Generate connect ID via `Protocol.generate_id()`
    - Encode connect command via `Protocol.encode_connect(id, opts)` and send via `Port.command(port, data)`
    - Wait for connect response with `receive` block. Timeout: `Keyword.get(opts, :login_timeout, 300_000)` (TRANS-04: 5 minutes for SSO)
    - On `{^port, {:data, data}}`: decode with `Protocol.decode_response(data)`. On `{:ok, ^id, _payload}` -> `{:ok, %State{...}}`. On `{:error, _, payload}` -> `{:stop, {:connect_failed, payload["message"]}}`.
    - On `{^port, {:exit_status, code}}` -> `{:stop, {:python_exit, code}}`
    - On timeout -> `Port.close(port)`, `{:stop, :connect_timeout}`

    **handle_call({:execute, sql, params, _opts}, from, state):**
    - Generate ID, encode via `Protocol.encode_execute(id, sql, params)`
    - `Port.command(state.port, data)`
    - `{:noreply, %{state | pending_request: {id, from}}}`

    **handle_call(:ping, from, state):**
    - Generate ID, encode via `Protocol.encode_ping(id)`
    - `Port.command(state.port, data)`
    - `{:noreply, %{state | pending_request: {id, from}}}`

    **handle_call(:disconnect, from, state):**
    - Generate ID, encode via `Protocol.encode_disconnect(id)`
    - `Port.command(state.port, data)`
    - `{:noreply, %{state | pending_request: {id, from}}}`

    **handle_info({port, {:data, data}}, state) -- response routing:**
    Extract `{expected_id, from}` or `{expected_id, from, :chunking, acc}` from `state.pending_request`.

    Decode data via `Protocol.decode_response(data)`.

    Match on response ID -- if it does not match expected_id, log warning and discard (`{:noreply, state}`).

    Response handling:
    - `{:ok, ^id, :chunked_start, payload}` -> Start accumulating. Set pending_request to `{id, from, :chunking, %{columns: payload["columns"], rows: [], total_rows: payload["total_rows"], metadata: payload["metadata"]}}`. Return `{:noreply, new_state}`.
    - `{:ok, ^id, :chunk, payload}` (when already chunking) -> Append payload["rows"] to acc.rows. Return `{:noreply, updated_state}`.
    - `{:ok, ^id, :chunk_done, _payload}` (when chunking) -> Build `%SnowflexDev.Result{columns: acc.columns, rows: acc.rows, num_rows: acc.total_rows, metadata: acc.metadata}`. `GenServer.reply(from, {:ok, result})`. Set pending_request to nil.
    - `{:ok, ^id, payload}` (non-chunked ok) -> Build `%SnowflexDev.Result{columns: payload["columns"], rows: payload["rows"], num_rows: payload["num_rows"], metadata: payload["metadata"]}`. `GenServer.reply(from, {:ok, result})`. Set pending_request to nil.
    - `{:error, ^id, payload}` -> Build `%SnowflexDev.Error{message: payload["message"], code: payload["code"], sql_state: payload["sql_state"]}`. `GenServer.reply(from, {:error, error})`. Set pending_request to nil.

    **handle_info({port, {:exit_status, code}}, state):**
    - If pending_request has a `from`, reply with `{:error, %SnowflexDev.Error{message: "Python process exited with code #{code}", code: "SNOWFLEX_DEV_EXIT"}}`.
    - `{:stop, {:python_exit, code}, %{state | port: nil, pending_request: nil}}`

    **terminate/2:**
    - If port is a port: send disconnect command via Protocol, then `Port.close(port)`.
    - Otherwise: `:ok`.

    **Private helpers:**
    - `query_timeout(opts)` -- `Keyword.get(opts, :timeout, 60_000)`
