    - run/1 raises Mix.Error with actionable message if python not found

    - run/1 prints success message with config example on completion
  </behavior>
  <action>
    Create `lib/mix/tasks/snowflex_dev.setup.ex` implementing `Mix.Tasks.SnowflexDev.Setup`:

    1. `use Mix.Task`
    2. `@shortdoc "Set up SnowflexDev Python environment"`
    3. Module attributes: `@venv_dir "_snowflex_dev/venv"`, `@pip_package "snowflake-connector-python>=3.12,<5.0"`
    4. `@impl Mix.Task` `def run(_args)` calls: `find_python!/0`, `verify_python_version!/1`, `create_venv!/1`, `install_deps!/0`, `verify_install!/0`, then prints success message

    `find_python!/0`:
    - `System.find_executable("python3")` first, fallback to `System.find_executable("python")`
    - If neither found, `Mix.raise("Python not found!\n\nSnowflexDev requires Python 3.9 or later.\nInstall Python from https://python.org or via your package manager:\n\n  macOS:   brew install python\n  Ubuntu:  sudo apt install python3 python3-venv\n  Fedora:  sudo dnf install python3")`

    `verify_python_version!/1`:
    - `System.cmd(python, ["-c", "import sys; print(f'{sys.version_info.major}.{sys.version_info.minor}')"])`
    - Parse output, check major >= 3 and minor >= 9
    - If too old: `Mix.raise("Python #{version} found, but SnowflexDev requires Python >= 3.9")`
    - Print: `Mix.shell().info("  Found Python #{version}")`

    `create_venv!/1`:
    - If `File.dir?(@venv_dir)`, print "Venv already exists" and return
    - `System.cmd(python, ["-m", "venv", @venv_dir], stderr_to_stdout: true)`
    - Non-zero exit: `Mix.raise("Failed to create Python virtual environment:\n\n#{output}\n\nMake sure the `venv` module is available:\n  Ubuntu/Debian: sudo apt install python3-venv")`

    `install_deps!/0`:
    - `pip = Path.join([@venv_dir, "bin", "pip"])`
    - `System.cmd(pip, ["install", "--upgrade", @pip_package], stderr_to_stdout: true)`
    - Non-zero exit: `Mix.raise("Failed to install snowflake-connector-python:\n\n#{output}\n\nIf behind a corporate proxy, try:\n  HTTPS_PROXY=http://proxy:port mix snowflex_dev.setup")`

    `verify_install!/0`:
    - `python = Path.join([@venv_dir, "bin", "python3"])`
    - `System.cmd(python, ["-c", "import snowflake.connector; print(snowflake.connector.__version__)"], stderr_to_stdout: true)`
    - Success: print version. Failure: `Mix.raise("snowflake-connector-python import failed after install: #{output}")`
